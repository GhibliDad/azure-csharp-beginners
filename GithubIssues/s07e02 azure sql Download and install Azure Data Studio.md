Azure Data Studio is a tool from Microsoft that can be used to interact with Azure SQL databases. Common uses include creating, modifying tables, manually running T-SQL queries to get, update, insert, delete data etc.

Azure Data Studio:
https://docs.microsoft.com/en-us/sql/azure-data-studio/download-azure-data-studio?view=sql-server-ver15

### Goal
* Azure SQL
* Azure Data Studio

### Steps
1.  Download and install Azure Data Studio
2. Create a new connection and enter relevant values
    - Server name can be copied from the Azure SQL Database resource in Azure Portal
